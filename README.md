# RocketsRCool

Just starting a project that tracks flights of reused boosters.
Currently tracks
 - Block 5 Falcon 9 boosters
 - Electron
 - Dragon 2 Capusles

Plans to track
 - New Shepard
 - Starship
 - Space shuttle
 - Starliner 

All data as of 07/25/2021. Data should be updated every mission; or at least every month.
